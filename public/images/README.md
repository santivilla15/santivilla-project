# Imágenes de Animales

Esta carpeta contiene las imágenes de animales que se usan en la aplicación.

## Cómo añadir imágenes:

1. **Opción 1: Usar imágenes locales**
   - Descarga imágenes de animales de sitios como Unsplash, Pexels, o Pixabay
   - Nombra las imágenes descriptivamente (ej: `animal-1.jpg`, `animal-2.jpg`)
   - Colócalas en esta carpeta

2. **Opción 2: Usar URLs de imágenes (recomendado)**
   - Usa servicios como Unsplash que proporcionan URLs directas
   - Las imágenes se cargan desde internet (no ocupan espacio en tu proyecto)
   - Mejor para desarrollo rápido

## Imágenes sugeridas:

Para el diseño de Santivilla, recomendamos:
- Perros y gatos en refugios (tema principal)
- Imágenes tiernas pero que muestren la necesidad
- Formato: JPG o WebP para mejor rendimiento
- Tamaño recomendado: 800x600px o superior

## Servicios gratuitos de imágenes:

- **Unsplash**: https://unsplash.com/s/photos/shelter-animals
- **Pexels**: https://www.pexels.com/search/dog/
- **Pixabay**: https://pixabay.com/images/search/animals/
